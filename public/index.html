Introduction

This API is used for Phonebook Application made by angularjs available at  Github Repository.

Register User

HTTP Method: POST
HTTP Request: /api/users/
Description: To register a new user using name, email and password.
Request Headers: 

                                        Content-Type : application/json

                                        x-auth-token : Not Required
                                        
                                    

                                

                                
 Request Parameters:
                                    

                                        

                                        
                                        No request parameter required
                                        
                                    

                                

                                
 Request Body:
                                    

                                        
{ "name":"String","email": "String","password": "String" }

                                    

                                

                                
 Response:
                                    

                                        
{"_id":"ObjectID", "name":"String", "email":"String","image":"data/base64"}
